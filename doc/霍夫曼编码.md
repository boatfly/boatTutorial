# 霍夫曼编码 Huffman Coding
- 应用于数据文件压缩
- 可变字长编码VLC的一种

## Huffman Tree 赫夫曼树
WPL最小的树，带权路径长度最短的树，权值较大的节点离根最近。
- 路径
- 路径长度
- 节点的权 值
- 带权路径长度 L-1
- 节点的带权路径长度：路径长度x节点的权
- 树的带权路径长度WPL weighted path length :所有的叶子节点的带权路径长度之和

## 霍夫曼编码的原理剖析
输入：用于传输的字符串内容

过程：
- 计算各个字符出现的次数
- 构建赫夫曼树
  - 从小到大进行排序，
- 给各个字符规定编码（前缀编码，避免避免编码重复），向左的路径为0，向右的路径为1
- 根据新的编码规范将发送的内容重新编码
